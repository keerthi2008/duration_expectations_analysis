# duration_expectations_analysis
duration and expectation analysis using pivot, radar chart in excel
##Project Overview :

This project was completed as part of the Cognifyz Technologies Data Analytics Internship. The objective of the project is to analyze participants' investment behavior, savings objectives, investment preferences, and expectations using Microsoft Excel.
The dataset contains demographic details and financial investment choices of participants. By applying Excel data analysis techniques such as pivot tables and data visualization, meaningful insights were derived to understand investment trends and financial decision-making patterns.

##Dataset Features
The dataset includes the following attributes:
Gender
Age
Investment Avenues
Mutual Funds
Equity Market
Debentures
Government Bonds
Fixed Deposits
Savings Objectives
Investment Duration
Investment Monitoring
Expected Returns
Reasons for choosing different investment options
Sources of financial information

##Tools & Techniques Used :

Microsoft Excel
Pivot Tables
Data Cleaning and Data Summarization
Data Visualization
Charts and Graphs

##Project Tasks :
Task 1: Gender Distribution Analysis
Analyzed the demographic distribution of participants based on gender. Visualization techniques were used to understand the representation of male and female investors.

Task 2: Investment Preferences Analysis
Examined participants' preferred investment avenues such as mutual funds, equity markets, government bonds, debentures, and fixed deposits. Pivot tables and bar charts were used to visualize the distribution of investment choices and reasons behind these preferences.

Task 3: Savings Objectives and Information Source Analysis
Analyzed participants' savings goals and the sources they rely on for financial information. Donut charts and horizontal bar charts were used to represent the distribution of savings objectives and information sources.

Task 4: Duration and Expectations Analysis
Studied the investment durations mentioned by participants and their expectations from investments. Histograms and radar charts were created to visualize investment duration patterns and compare investment expectations.

##Key Insights :

Identified the most preferred investment avenues among participants.

Analyzed the major savings objectives influencing financial planning.

Observed common sources of financial information used by investors.

Studied how investment duration relates to expected returns and investment goals.

##Conclusion :
This project demonstrates practical application of Excel for financial data analysis. Through pivot tables and data visualizations, the project highlights patterns in investment behavior and helps in understanding participants' financial planning strategies.
